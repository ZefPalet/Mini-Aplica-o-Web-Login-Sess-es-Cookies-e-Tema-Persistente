# Mini-Aplica-o-Web-Login-Sess-es-Cookies-e-Tema-Persistente
Mini aplicação web desenvolvida com HTML, CSS e JavaScript puro, incluindo tela de login com validação, gerenciamento de sessão via sessionStorage, personalização de tema claro/escuro usando cookies e proteção simples de rota. Projeto demonstra conceitos essenciais de autenticação e persistência no front-end.
